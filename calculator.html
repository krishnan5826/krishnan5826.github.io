Operators Handbook Supported Features Features not supported
Common operators:
+ : computes the sum of any two numbers
- : computes the difference between any two numbers
* : Multiplies any two numbers
/ : Divides any two numbers
Single digit computations: like
8 + 2
8 - 2
8 * 2
8 / 2
 Order of Operations:
This JavaScript Calculator computes as the numbers are clicked.
For example, the expression 2 + 4 * 3 will give a result of 18 and not 14
Special operations:
^ : Computes exponent
√ : Computes the square root
π : Substitues the value of pi (3.14159)
e : Substitues the value of Euler's Number (2.71828)
sin : computes the sine of the entered number (in radians)
cos : computes the cosine of the entered number (in radians)
tan : computes the tangent of the entered number (in radians)
log : computes a logarithmic function for the entered number (in radians)
Abs : computes the absolute value of the entered number (in radians)
Rnd : Rounds up/down the number
CLR : Clears values and resets the calculator
<<< : Backspace, removes the last entered value
