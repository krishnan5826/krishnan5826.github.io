Operators handbook Supported Features Features not supported
Common operations:
+ : computes the sum of any two numbers
- : computes the difference between any two numbers
* : Multiplies any two numbers
/ : Divides any two numbers
Single digit computations like
8 + 2
8 - 2
8 * 2
8 / 2
 Order of Operations
The javascript calculator computes as the numbers are typed.
For example, the expression 2 + 4 * 3 will give a result of 18 and not 14
Special operations:
^ : Computes exponent
√ : Computes the square root
π : Substitues the value of pi
Rnd : Rounds up/down the number
CLR : Clears values and resets the calculator
<<<< : Backspace, removes the last entered value

Multi digit computations like
286 + 24
84 - 62
28 * 22
88 / 11
1 + 22 + 333 + 4444 		Evaluating negative expressions
The code does not differentiate between a -ve sign and the -ve operator.
While -88 + 11 and -88 -11 will work, 88 + -11 and -88 * -11 will not work and will give unpredictable results.
